- Take strings of parentheses from the user as input and use a stack to
  check if the string of parentheses is balanced or not.

    - Input: ()
    - Output: Balanced
  
  - Test case 2:
    - Input: (())()(()())()
    - Output: Balanced
   
  - Test cse 3:

    - Output: Not balanced
